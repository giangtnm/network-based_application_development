Câu 1

Nhúng vị trí nào có thể đặt Javascript trong file HTML và đặt theo cú pháp nào? Nên đặt Javascript tại vị trí nào để có hiệu suất tốt nhất? Cho ví dụ minh họa.

Javascript có thể đặt ở phần head và phần body. Javascript được đặt theo cú pháp <script type="text/javascript"></script>. Nên đặt Javascript tại vị trí cuối phần body để đạt hiệu suất tốt nhất vì nếu đặt ở phần đầu body hay ở phần head thì nếu Javascript bị lỗi file .html sẽ bị chậm

- Trong <head>:
.....
<head>
  <script>
  function myFunction() {
    document.getElementById("demo").innerHTML = "Paragraph changed.";
  }
  </script>
</head>
<body>
.....

- Trong <body>:
.....
<body>
  <script>
  function myFunction() {
    document.getElementById("demo").innerHTML = "Paragraph changed.";
  }
  </script>
</body>
</html>
.....

- JavaScript được đặt từ file bên ngoài
<script src="myScript1.js"></script>

- JavaScript được đặt từ nguồn bên ngoài
<script src="https://www.w3schools.com/js/myScript1.js"></script>

Câu 2

Trong Javascript có những kiểu dữ liệu nào? Làm thế nào để tạo một biến để lưu trữ giá trị tương ứng với những kiểu dữ liệu đó?

Trong Javascript không phân biệt kiểu dữ liệu của biến. Khi khai báo biến chỉ cần khai báo từ khóa var.

Câu 3

Làm thế nào để truy xuất (lấy giá trị) trong 1 thẻ HTML bất kỳ và gán vào 1 biến để trong Javascript theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector (ví dụ .class, #id)? Cho ví dụ minh họa.

  • Theo ID: document.getElementByID()
    VD: var element = document.getElementById('id');
  • Theo Class: document.getElementsByClassName()
    VD: var element = document.getElementsByClassName('class');
  • Theo Name: document.getElementsByName()
    VD: var element = document.getElementsByName('name');
  • Theo tên thẻ HTML: document.getElementsByTagName()
    VD: var element = document.getElementsByTagName('tag');
  • Theo CSS Selector: document.querySelectorAll()
    VD: var element = document.querySelectorAll("selector.css");
